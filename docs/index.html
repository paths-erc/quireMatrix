quireMatrix by PAThs

An easy to use, dependency-less software to visualise in the browser the layout of quires, using a very intuitive notation system.

Feel free to download and use and it; a credit or reference to PAThs project http://paths.uniroma1.it/ will be much appreciated :)

The full documentation on library and notation system is available on Github
